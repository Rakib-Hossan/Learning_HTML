HTML Comment

To comment something or hide something on html we use comment. Syntax: <!-- content -->

HTML Links

To define a link in HTML we use <a> tag with href attribute. Syntax: <a href="links"> content </a>

HTML Image

To show image in html we use <img> tag with src and alt attribute. Where srcattribute value is source of image and altattribute value alternative text of image. Syntax: <img src="source of image" alt="alternative text shown if the image doesn't load on browser.">

HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab. We set a favicon using link tag.

Example:

<link rel="shortcut icon" href="source of favicon" type="image/x-icon">

HTML Table

To create a table in HTML we write <table> tag. To create row we write <tr>tag. To create column we write <td>tag. To set heading on table we write <th>tag.

Example:

Name Age Hobby
Rakib 26 Coding
Akib 27 Traveling
Shakib 26 Singing

Html colspan and rowspan

if we marge column in table we write colspan attribute and if we marge row in table we write rowspan attribute.

Example of colspan
