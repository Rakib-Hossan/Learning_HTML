HTML Comment

To comment something or hide something on html we use comment. Syntax: <!-- content -->

HTML Links

To define a link in HTML we use <a> tag with href attribute. Syntax: <a href="links"> content </a>

HTML Image

To show image in html we use <img> tag with src and alt attribute. Where srcattribute value is source of image and altattribute value alternative text of image. Syntax: <img src="source of image" alt="alternative text shown if the image doesn't load on browser.">
