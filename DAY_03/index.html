HTML Block Elements

A block element always takes up the full width available.

HTML div element

In html we used <div> tag as a container. It is a block element.

Example of div: To see div is a block element we style div tag by border.

HTML inline Elements

An inline element only takes up as much width as necessary.

HTML span element

In html we used <span> tag as an inline container to mark up a part of a text, or a part of a document. It is an inline element.

HTML class attribute

The HTML class attribute is used to specify a class for an HTML element. Multiple HTML elements can share the same class. syntax: <TagName class="class_name">. We also use Multiple class in same tag. Syntax for that: <TagName class="class_name1 class_name2 ">

HTML id Attribute

The HTML id attribute is used to specify a unique id for an HTML element. We cannot have more than one element with the same id in an HTML document. Syntax:<TagName id="Id_name">.
