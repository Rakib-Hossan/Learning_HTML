HTML LIST

There type list we used in html. 1. Ordered list, 2. Unordered list and 3. Description list

An ordered list can be numerical or alphabetical. To define ordered list we use <ol>tag.

Example of Ordered List:

Default marking start with numerical

  1. 1st
  2. 2nd
  3. 3rd
  4. 4th

We can change the list item marker using type attribute.

Example:

Marking Start with A

  1. first
  2. second
  3. third

Marking Start with Roman i

  1. first
  2. second
  3. third

To define unordered list we use <ul>tag.

Example of Ordered List:

Example:

Marking style is circle
