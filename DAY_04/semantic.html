Semantic Element

Semantic elements means element with a meaning. There are some semantic elements are: article; aside; details; figcaption; figure; footer; header; main; mark; nav; section; summary; time.

HTML section element

Section element defines a section in a document.

HTML article element

Article element defines specifics independent, self contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.

HTML header element

Header element represents a container for introductory content or a set of navigational links. Header element typically contain: 1. heading 2. logo or icon 3. authorship information.

HTML footer element

Footer element defines a footer for a document or section. Footer element typically contains: 1. authorship information 2. copyright information 3. contact information 4. sitemap 5. back to top links 6. related documents

HTML nav element

nav element defines a set of navigation links.

HTML aside element

Aside element defines some content aside from the content it is placed in.

HTML figure element

figure element specifies self-contained content, like illustrations diagrams, photos, code listings etc.

HTML figcaption element

figcaption tag defines a caption for a figure element.
